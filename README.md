## Dimmed One Dark Syntax theme [extended and fixed]

![one-dark-syntax](https://cloud.githubusercontent.com/assets/378023/7783203/49271ef6-0174-11e5-8eb1-d80b8d34a0e3.png)


There is also a matching [UI theme](https://atom.io/themes/one-dark-ui).

### Install

This theme is installed by default with Atom and can be activated by going to the __Settings > Themes__ section and selecting it from the __Syntax Themes__ drop-down menu.


### Extended?
Syntax highlight added for:
* PHP
* XML

Syntax highlight fixed for:
* SCSS
* HTML
* JS
